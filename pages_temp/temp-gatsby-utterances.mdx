## utterance란?
[https://utteranc.es/](https://utteranc.es/)

* github issue 연동 댓글 github app
* free to use! (광고 없음)
* **theme으로 preferred color scheme 제공**

preferred color scheme은 사용자의 모바일 혹은 데스크톱 환경(다크, 라이트 모드)에 따라 다크/라이트 모드를 변환할 수 있는 기능이다.
이 블로그 자체에도 적용되어 있어 다크/라이트 모드 전환 시 블로그도 마찬가지로 다크/라이트 모드로 적용된다.
